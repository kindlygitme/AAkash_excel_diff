# Aakash ExcelDiff Pro

Excel file comparison tool — cell-by-cell differences across all sheets.

## Features
- 🟡 Updated cells detection
- 🔴 Deleted rows detection  
- 🟢 Added rows detection
- 📄 Multi-sheet support
- 📊 Master report download
- 🔆 Highlighted output file
- 📧 Email report (Gmail)

## Run Locally

```bash
pip install -r requirements.txt
streamlit run exceldiff_aakash.py
```

## Deploy on Streamlit Cloud
1. Push this repo to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Connect GitHub → select this repo → `exceldiff_aakash.py`
4. Deploy!
